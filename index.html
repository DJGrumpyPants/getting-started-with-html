Things to Remember


HTML - Hyper Text Markup Language This is the main doccuemtn type of the web. It can contain Text content, markups, and refrences

There are four pieces involved in browsing the internet:
1. Browsers - This is a program run on your local computer. It conencts to remote sources and displays files from remote locations
2. The internet - This is the magical cloud of wires and conenctions that gets your browser to the servers.
3. Markups - these modify the way text appears on the page. Some to remember are:
b = Bold
u = underline
br = line break
p = paragraph
em = emphasis (italics)
a = anchor - an anchor is used with attributes such as href.

Markups can be inline like br, or block like p.
Inline markups modify a piece of text
Block markups put a box around whatever is contained inside the marks.
So a p (paragraph) would be contained inside a small box, and a piece of text that is made bold is inline.
